# kalc
scamming calculator. SCAMMING DEMONSTRATION

## QUICK LAUNCH
1. ``cd kalc-1.0.0``
2. ``gedit kalc``
3. search for the function ``cmd`` to change the ip address
4. ``sudo chmod 777 kalc``
5. ``sudo ./kalc``


## SETUP

1. ``cd kalc-1.0.0``
2. ``gedit kalc``
3. search for the function ``cmd`` to change the ip address
4. ``sudo chmod 777 build``
5. ``sudo ./build``
6. Run ``dpkg -i kalc-1.0.0.....deb`` in terminal to install the app.
